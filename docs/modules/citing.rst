Citing sfsutils
===============

``sfsutils`` was created as part of the :raw-html:`<a href="https://fastdfe.readthedocs.io"><code class="docutils literal notranslate">fastdfe</code></a>` project. If you use ``sfsutils`` in your research, please consider citing the following paper, as part of which it was originally developed:

.. code-block:: bibtex

    @article{fastdfe,
        author = {Sendrowski, Janek and Bataillon, Thomas},
        title = {fastDFE: fast and flexible inference of the distribution of fitness effects},
        journal = {Molecular Biology and Evolution},
        publisher = {Oxford University Press},
        pages = {msae070},
        year = {2024},
        month = {04},
        issn = {1537-1719},
        doi = {10.1093/molbev/msae070},
        url = {https://doi.org/10.1093/molbev/msae070}
    }
